## HTML INDEX
```
<input type="button">	Botón que no envía el formulario, se usa para acciones personalizadas con JavaScript.
<input type="checkbox">	Casilla de verificación para seleccionar o no una opción.
<input type="color">	Selector de color que permite elegir un color desde una paleta.
<input type="date">	Selector para ingresar una fecha (solo día, mes, año).
<input type="datetime-local">	Selector de fecha y hora (sin zona horaria).
<input type="email">	Campo para ingresar correos electrónicos. Valida formato de correo.
<input type="file">	Permite subir archivos desde el dispositivo.
<input type="hidden">	Campo oculto, no visible para el usuario pero su valor se envía con el formulario.
<input type="image">	Permite enviar el formulario mediante una imagen como botón.
<input type="month">	Selector para elegir solo mes y año.
<input type="number">	Campo para ingresar números, con opciones de mínimo, máximo y paso.
<input type="password">	Campo para contraseñas; oculta los caracteres escritos.
<input type="radio">	Botón de opción única dentro de un grupo (solo uno se puede seleccionar).
<input type="range">	Barra deslizante para seleccionar un valor dentro de un rango.
<input type="reset">	Botón que reinicia todos los campos del formulario a su valor inicial.
<input type="search">	Campo de búsqueda con estilo y comportamiento específicos.
<input type="submit">	Botón para enviar el formulario.
<input type="tel">	Campo para ingresar números telefónicos.
<input type="text">	Campo básico de texto para cualquier dato.
<input type="time">	Selector de hora (sin fecha).
<input type="url">	Campo para ingresar direcciones web. Valida formato de URL.
<input type="week">	Selector para elegir una semana de un año.
required:	Hace obligatorio completar ese campo antes de enviar el formulario.
placeholder:	Texto de ayuda que aparece dentro del campo hasta que el usuario escribe algo.
autofocus:	Coloca el foco automático en ese campo al cargar la página.
pattern:	Expresión regular que define el formato permitido para el valor del campo.

div class="" Para asignarle un nombre de clase a ese div y luego poder aplicarle estilos CSS
